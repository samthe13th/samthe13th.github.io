Web Development

Palette Machine

A JavaScript color palette maker for designers


www.palettemachine.com

Tools used:

Drag and drop UI

I wanted to make something that was playful, intuitive, and gave me the feeling of working with a physical palette, like a painter.

In any design, one of first questions I try to answer is: "How does the user want this to work? What feels natural?"

Color Blending

Often when designing a color palette, it is useful to set up cornerstone colors first, and then find intermediates between these.

To support this, I designed an easy way to blend colors on the palette, by simply clicking between two colors to mix them.

There is also a separate "Blender"" that can be used to mix two colors to find an intermediatory, which can then be dragged onto the palette.

Reference Images

Often when designing a website or logo, it is often important to make a palette that fits existing properties.

For this reason, I wanted to give the user the opertunity to easily grab colors from a reference image.

JavaScript Apps


DEMO

This is a small web app I created for UBC's Physics 100 course.

My intent was to make something that would allow students to practice modeling freebody diagrams (a common task in first year physics).

The app written with JavaScript, html, and css, with some help from the HTML5 framework Phaser.io.

Interactive Graphics


DEMO

You might have noticed that I like sliders.

One great thing about range sliders is that they can allow the user to step through the frames of an animation at their own pace.

This can be a really powerful way of communicating processes that change with time.

Simulations


DEMO

Another example of using a slider, but with some extra interactive elements.

This projectile motion simulator allows students to "throw" a ball and track its various properties as it falls.

My goal was to give students a way to virtually test their predictions and assumptions regarding projectile motion.
