Interactive Web Design

Palette Machine

A javascript color palette maker for designers

Drag and drop UI

I wanted to make something that was playful, intuitive, and gave me the feeling of working with a physical palette, like a painter.

In any design, one of first things I ask myself is: 'How does the user want this to work?

What feels natural?'

Color Blending

Often when designing a color palette, it is useful to set up cornerstone colors first, and then find intermediates between these.

To support this, I designed an easy way to blend colors on the palette, by simply clicking between two colors to mix them

Reference Images

My, what a beautiful Scottish sheep! (Photo credit my mother)

Often when designing a website or logo, it is important to make a palette that fits existing properties. For this reason, I knew it would be important for Palette Machine to be able to build palettes using reference photos
